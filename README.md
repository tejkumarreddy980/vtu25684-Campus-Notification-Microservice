# Campus Notification Microservice

A responsive frontend interface built with React and Vite to manage, broadcast, and track real-time campus notifications. This service ensures seamless communication across departments, faculty, and students.

## 🚀 Features

* **Real-Time Alerts:** Immediate delivery of critical campus announcements and emergency notifications.
* **Targeted Broadcasting:** Ability to filter and send updates to specific departments, batches, or user roles.
* **Modern UI/UX:** Fast, minimalist user interface optimized for both desktop and mobile devices.
* **High Performance:** Powered by Vite with Hot Module Replacement (HMR) for an ultra-fast development experience.

## 🛠️ Tech Stack

* **Frontend Framework:** React 19
* **Build Tool:** vscode
* **Linting & Quality:** ESLint
